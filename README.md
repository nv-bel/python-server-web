# Python Server Web

## Project Overview

This project is a web server built using Python that serves static web pages and provides a RESTful API for handling dynamic requests. It aims to demonstrate the use of Python's web frameworks and is suitable for both learning and practical use.

## Features
- **Static File Serving**: Automatically serves HTML, CSS, JavaScript, and images.
- **RESTful API**: Handles GET, POST, PUT, and DELETE requests.
- **User Authentication**: Supports basic user authentication for API endpoints.
- **Logging**: Logs incoming requests to monitor activity and debug issues.
- **Configuration**: Easily configurable settings through environment variables.

## How to Run
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/nv-bel/python-server-web.git
   ```
2. **Navigate to the Directory**:
   ```bash
   cd python-server-web
   ```
3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the Server**:
   ```bash
   python app.py
   ```
5. **Access the Server**:
   Open your web browser and navigate to `http://localhost:5000`.

Enjoy using the Python server and customize it as per your needs!
